Game economy

The MonkeyLeague game economy is centered around an in-game currency, MonkeyBucks $MBS, and in-game assets, MonkeyPlayer NFTs, that hold value in and out of the game.

Get $MBS from exchanges, use it for game features and in-game items, and earn it through gameplay.
